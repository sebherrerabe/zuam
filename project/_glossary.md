# Glossary

| Term | Meaning |
|---|---|
| Zuam-owned fields | Task metadata stored only in Zuam and never synced back to Google Tasks as authoritative source data |
| Google-owned fields | Task existence and synced task fields where Google Tasks is the system of record |
| Vertical slice | A planning and implementation unit that includes requirements, contracts, tests, and delivery boundaries for one capability |
| Work packet | The LLM-ready implementation brief for one module or sub-slice |
| Smart List | A system-generated task collection such as Today, Next 7 Days, Inbox, Completed, Won't Do, or Trash |
| Section | An app-side grouping within a list that doubles as Kanban columns when relevant |
| Nudge level | The escalation tier of reminder behavior, from ambient through nuclear |
| Phase-ready | A phase whose critical-path modules are documented well enough to begin implementation |
| Decision-complete | A document state in which an implementer can proceed without needing new product or architecture choices |
